#Stopwatch and Timer

<ul>
  <li>Start</li>
  <li>Stop</li>
  <li>Reset</li>
</ul>
<br>
A Countdown <b>Timer</b> is also being made as a part of this project, which is set for the event of Christmas and it restarts again once the event occurs, for the same event in upcoming year.
